# SR 11-7: Plain-English Explainer

> Status: Placeholder — populated alongside Phase 3

SR 11-7 (Supervisory Guidance on Model Risk Management, Federal Reserve, 2011) establishes
the framework US banks use to govern quantitative models. Key pillars:

1. **Model Development** — rigorous methodology, documented assumptions, out-of-sample testing
2. **Model Validation** — independent challenge of conceptual soundness, data quality, outcomes
3. **Ongoing Monitoring** — performance tracking, population drift, periodic review
4. **Governance** — inventory, tiering by risk, board/management oversight

This project demonstrates compliance-awareness across all four pillars.
