# Data

## Source

IEEE-CIS Fraud Detection dataset — Vesta Corporation, hosted on Kaggle:
https://www.kaggle.com/competitions/ieee-fraud-detection/data

## Setup

1. Accept the competition rules on Kaggle.
2. Download the four CSV files:
   - `train_transaction.csv`
   - `train_identity.csv`
   - `test_transaction.csv`
   - `test_identity.csv`
3. Place them in `data/raw/`.

These files are gitignored and must never be committed.

## Processed outputs (`data/processed/`)

Generated by `src/data/preprocess.py`. Also gitignored.

| File | Description |
|------|-------------|
| `joined.parquet` | Merged transaction + identity table |
| `train.parquet` | Training split (first 80% by `TransactionDT`) |
| `val.parquet` | Validation split (last 20% by `TransactionDT`) |
| `preprocessor.joblib` | Fitted column transformer |
